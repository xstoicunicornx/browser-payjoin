# Browser Payjoin

All this does is print true to the console to confirm the sender session was saved and advanced to the next state.

## Usage

```bash
# install packages
npm install

# modify entry file
bash scripts/vite_wasm_import_mod.sh

# run dev server
npm run dev
```

## Notes

For payjoin, the NPM package @xstoicunicornx/payjoin was used. This package was built using [Uniffi Bindgen React Native](https://jhugman.github.io/uniffi-bindgen-react-native/reference/config-yaml.html#web) `target: web` option.

The `vite_wasm_import_mod.sh` script modifies the `.wasm` import to be compatible with Vite and adjusts the file directories to use the current `dist` directory rather than the `src` directory that was autogenerated (I have no clue why `index.web.js` references the `src` directory, its the only one that does)
